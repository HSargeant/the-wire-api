The Wire API Documentation

The API for the seminal TV series. The API contains data for of characters, quotes and character deaths. With episode data coming soon. This documentation will help with all you need to begin making your request.

Base Url

https://the-wire-api.herokuapp.com/api

Character Endpoints

Character Attributes

Attribute Type Description
char_id string Unique id per character
name string A character's full name
occupation array List of known occupation/s
image string Character's image
status string Is the character dead or alive
aka string A nickname or alias the character is referred as
seasons array List of seasons that the character appeared in
about string Brief character bio

Get all characters

Endpoint to retrieve information from all characters.

/api/characters

Get single character

Example request by character id.

/api/characters/2

          {
            "char_id": "2",
            "name": "Avon Barksdale",
            "occupation": ["Head of the Barksdale Organization"],
            "seasons": [1,2,3,5],
            "DOB": "August 15, 1970",
            "status": "Alive",
            "actor": "Wood Harris",
            "image": "https://images4.imagebam.com/31/85/39/MEBDII1_o.jpg",
            "category": "The Street",
            "about": "Leader of the drug organization that bears his name, 
                      Barksdale grew up in the Terrace high-rises and has managed to avoid arrest, 
                      remaining a furtive but increasingly powerful force in the Westside drug trade.",
            "aka": []
          }

Get characters by category

Request characters by category: 'Street', 'Law', 'Port', 'City+Hall', 'The Next Generation (next)', or 'News'

/api/category/street

Get random character

Request a random character

/api/characters/random

Search character by name

/api/characters/d'angelo

This works with the first name, last name or nickname.

Quote Attributes

Attribute Type Discription

/api/quotes/random

Character Deaths Endpoints

Death Attributes

Attribute Type Description
name string The name of the deceased character
cause string How the character died
responsible string The person(s) is directly responsible for the death
season string The season the death occured in
episode string The episode of the season

Get all death information

Endpoint to retrieve information about every death in Breaking Bad.

/api/deaths

Get a character's death information

Endpoint to retrieve information about an individuals death

/api/death/omar

Get a list of deaths caused by a character.

Use this enpoint to find out how many bodies a character has from the series.

api/deaths?responsible=wee-bey

Just as for characters the parameters work for nick names as well as first and last names.

Get random death information about a character

/api/random-death
